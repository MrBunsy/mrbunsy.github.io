2D Engine Version 4

Development with version 3 of the 2D engine and the sandbox ground to a halt a while ago. A year on from the last time I touched the engine and I've majorly re-written it for version 4! Stable collisions between polygons are now working, and I've just got stable springs working. I'm currently working on string which can be properly collided with.

Colliding polygons

Sandbox 4 will be in development shortly and more demos will be posted here as I finish them! This version is fairly properly object orientatedly abstracted and so should be very easy to embed in other projects, something which was never possible with the old versions of the engine.
