How to Implement

Add a span with the id 'windowspan'. 
<span id="windowspan"></span>
Include the divwindows.js script. 
<script src="divwindows.js" type="text/javascript"></script>

How to Use

Create a divwindow object in javascript 
var testwindow = new divwindow('testwindow','Testing',1,'white','yellow',300,200,300,300,3,'This is a test window.  Wowee.','')

divwindow() accepts the following parameters:

new divwindow(name - string must be same as object referer,
title - string,
zindex - interger,
background colour - string,
title colour - string,
width - integer,
height - interger,
left - integer,
top - integer,
padding - integer,
contents - string,
javascript to execute when window shuts - string)

Contents of window can be changed like this: 

document.getElementById(testwindow.content).innerHTML='New Contents';

Known Bugs

In IE if the window isn't open itially the border won't be rendered after the window is opened until the window is moved. However, I think this can be forced by moving the div just using javascript after the window has been created.
