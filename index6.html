The stat counter now produces fancy looking pie charts, thanks to an anti-aliasing script I found.

Also, the Graph Plotter has had anti-aliasing added as an option hidden in advanced (Thanks to this script). However, the quality isn't particularly great as some lines aren't improved much. f(x) (green) is much better, however the polar (blue) has barely changed.

New News System!

I've brought the news/blog/comic system I wrote for www.squinkles.co.uk over here at last. This makes it much easier for me to add news, and includes an automatic archive and RSS system!

This system is unlikely to ever become open source because it requires a bit of editing to be integrated into an existing site, and probably doesn't have all the features someone would look for in setting up a site. Also, there are better and far more mature systems out there.

I'm also slowly adding all the old news items to build up the archive.
